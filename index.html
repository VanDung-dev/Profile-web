Game Cờ Vua Python
Ứng dụng game cờ vua đa nền tảng sử dụng PyGame với giao diện đồ họa 2D cơ bản. Hỗ trợ cả chế độ chơi hai người và chơi với máy ở nhiều cấp độ khó. Tính năng nổi bật: Gợi ý nước đi bằng thuật toán Minimax, undo/redo, và mô hình cờ vua tự phát triển không dựa trên PyChess.

Python Chess Game
A cross-platform chess game built with PyGame, featuring a basic 2D graphical interface. Supports both two-player mode and AI opponents at various difficulty levels. Key features: Move suggestions using the Minimax algorithm, undo/redo, and a custom chess model not based on PyChess.
